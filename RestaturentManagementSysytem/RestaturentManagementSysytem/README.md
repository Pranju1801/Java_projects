
## 📌 Restaurant Management System

### 📖 Project Description

Restaurant Management System is a web-based application that allows administrators and restaurant owners to manage multiple restaurant branches, maintain restaurant data, and handle operations efficiently.

-----------------------------------------------------------------

### 🚀 Features

* Admin login authentication
* Add / Update / Delete restaurant branches
* Manage restaurant details
* Database integration using MySQL
* Exception handling
* MVC architecture implementation

-------------------------------------------------------------------

### 🛠 Tech Stack

* Java
* Spring MVC
* Hibernate (ORM)
* MySQL
* JSP
* HTML, CSS
* Apache Tomcat
