# Bank Application

## Overview
The Bank Application is a web-based project that allows users to perform basic banking operations online. It provides features such as account registration, login, deposit, withdrawal, and account management in a secure and simple interface.

## Features
- User Registration
- User Login and Authentication
- Deposit Money
- Withdraw Money
- Delete Account
- View Account Details

## Technologies Used
- Backend: Java,Spring MVC, Hibernate,Exception Handling 
- Frontend: JSP, HTML, CSS
- Database: MySQL

## Project Architecture
This project follows the MVC (Model-View-Controller) architecture:
- Model: Handles database operations using Hibernate
- View: JSP pages for user interface
- Controller: Spring MVC controllers to manage requests and responses

## Installation and Setup

1. Clone the repository
2. Open the project in IDE (Eclipse / IntelliJ)

3. Configure MySQL Database
- Create a database in MySQL
- Update database username and password in the configuration file

4. Run the project on Apache Tomcat Server

5. Open in browser

## Project Modules
- Registration Module
- Login Module
- Transaction Module (Deposit & Withdrawal)
- Account Management Module
